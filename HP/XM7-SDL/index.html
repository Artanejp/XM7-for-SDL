インストール:

- ソースからビルドする場合は、make ; sudo make install　で大丈夫になりました(0.2β以降)。
但し、libagarのソースやフォントはバンドルされていません。
- AGAR1.4.1以降とOpenGL,SDL,SDL_Mixerが実行に必要です。
(XM7バイナリにはAgarバイナリを添付してあります。) - 自前でビルドする場合 1.4.1SVN9049以降を推奨します (r9049のソースコード)
- IPA Gothic(V3)が必要です。システムにないときは(Linux : $HOME/.xm7 / Win32 カレントディレクトリ/xm7 )に置いてください(NEW!)
- 起動にはロムが必要です。(Linux : $HOME/.xm7 / Win32 .\xm7)にコピーして下さい - bin/xm7 と bin/xm7.debug を/usr/local/bin などにinstall
- src/resource/ 以下とlocale以下をを /usr/local/share/xm7 にインストール
- sudo ldconfigする(必ずやること)
