インストール:

- ソースからビルドする場合は、make ; sudo make install　で大丈夫になりました(0.2β以降)。但し、libagarのソースやフォントはバンドルされていません。
- AGAR1.4.1以降とOpenGL,SDL,SDL_Mixerが実行に必要です。(XM7バイナリにはAgar バイナリを添付してあります。)自前でビルドする場合1.4.1SVN8929以降を推奨します(r8929のソースコード)
- IPA Gothic(V3)が必要です。システムにないときは$HOMEかカレントディレクトリィに置いてください(NEW!)
- bin/xm7 と bin/xm7.debug を/usr/local/bin などにinstall
- resource/ 以下を /usr/local/share/xm7 にインストール
- libagar.bin/ 以下を/usr/local あたりにInstall (※ Agarとのヴァージョン整合性の問題があるので必ずやること)
- sudo ldconfigする
