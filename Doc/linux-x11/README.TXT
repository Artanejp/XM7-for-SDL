□FM-7 EMULATOR XM7 (V2.8L31a/V3.3L31a) For Linux + GTKとは

  このアーカイブはFM-7エミュレータXM7(V2.8L31a/V3.3L31a)をLinux環境で動作できる
  ようにするためのソースコードです。GTK+-2.0を使用しています。

  Linuxに限らず若干の修正でBSD系のOSでも動作するかもしれません。
  とりあえず動くようになったというレベルですのでこれから進化するかもしれません。

  めざすところはXM7 Ver1.0系のようにオフィシャルにLinux対応してもらうことでしょうか。

□使用規定
  アーカイブに含まれるファイル群はオリジナルのXM7(V2.8L31a/V3.3L31a)のwin32
  ディレクトリのソースを再利用して開発しました。また一部のソースはGIMONSオリジナル
  のソースコードも含まれます。

  オリジナルのソースコードはＰＩ．氏およびたけがみ りう氏が著作権を保有しております。

  オリジナルのソースコードの使用規定では再利用は自由/商用利用は禁止ですので、この
  アーカイブのソースコードも同様の規定となります。

  またこのアーカイブの再配布はできません。無断転載を禁じます。

□ファイル及びディレクトリ
  以下のようなファイルとディレクトリのみが含まれます。

  README.TXT  このファイル
  COMPILE.TXT コンパイル方法の説明テキスト
